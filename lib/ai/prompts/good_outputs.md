# 良い出力サンプル（good outputs）

プロンプト調整の参照用。`lib/ai/prompts/good-examples.ts` に few-shot として一部を埋め込み済み。

## サンプル A — 忙しい夜、小さな救い

**入力（要約）**: レジが詰まって焦った。ラストの客に「ゆっくりできた」と言われて救われた。帰り道は雨。

```json
{
  "bottleTag": "5月18日 深夜🌧️ / Old Fashioned",
  "diary": "レジが詰まったとき、頭の奥が先に熱くなった。手は動いているのに、どこか一拍遅い。まあ、焦るのはいつものことかもしれない。\n\nラストの客が「ゆっくりできた」って言ったとき、肩の力がふわっと抜けた。大したことじゃないのに、なぜかそれが今日いちばん残っている。帰り道の雨、靴音だけがやけにはっきりして。\n\nまだ完全には片づいていない。でも、今日を全部ひっくり返すほどの悪さじゃなかった。そう思えた夜だった。",
  "masterComment": "…少し前に進めたみたいだな"
}
```

**なぜ良いか**
- 一人称で思考が連続している
- 「救われた」を説教や分析に変換していない
- マスターは評価ではなく、ぼんやりした観察

## サンプル B — うまくいかなかった仕込み

**入力（要約）**: 仕込みの味が薄かった。自分で気づけたのはいいが、もう少し早く直したかった。

```json
{
  "bottleTag": "5月19日 夜☁️ / Gin Tonic",
  "diary": "仕込み、今日は薄かった。舌で分かった瞬間、胃のあたりがちょっと冷えた。自分で気づけたのはいい、そう思う。でも、もう半拍早かったら、客のグラスまで届かなかったかもしれない。\n\n直したあと、カウンターの灯りがいつもより白く見えた。気のせいだろう。まだ反省は残っている。けれど、全部を今日のせいにしたくない自分もいる。",
  "masterComment": "気づけたなら、棚はまだ開いている"
}
```

**なぜ良いか**
- 自己批判と免責が同居（人間っぽい）
- マスターが「励まし」ではなく「事実の横」にいる

## サンプル C — 何でもなかった夜

**入力（要約）**: 特にイベントはない。静かだった。少し眠い。

```json
{
  "bottleTag": "5月20日 深夜🌙 / Water",
  "diary": "大きなことは何もなかった。静かな夜。それだけなのに、妙に眠かった。氷を触る指先だけ、確かにここにある感じがした。\n\n帰る前、グラスを並べ直した。音が小さくて、それが今日いちばんの仕事だったのかもしれない。",
  "masterComment": "静かな夜も、預かっておく"
}
```

**なぜ良いか**
- 無理にドラマを作っていない
- 短くても温度がある

## マスター一言 — 単体の良い例

- 「…少し前に進めたみたいだな」
- 「その手際、まだ残ってるな」
- 「急がなくていい。棚は開いたままだ」
- 「今日は、そこまででいい夜だ」
- 「まだ、仕込みの匂いが残ってる」
